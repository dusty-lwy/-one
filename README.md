# 用户登录管理系统（面向对象重构版）

## 项目简介
本项目是一个基于 Python 开发的命令行用户登录管理系统，支持用户注册、登录、管理员权限管理等功能。项目从最初的“面条代码”版本，经过系统性重构，最终实现了一个职责清晰、可扩展的三层架构版本，体现了对面向对象设计和软件工程原则的理解。

## 版本演进

| 版本 | 文件名 | 特点 |
|---|---|---|
| V1.0 面条代码版 | `login_v1.py` | 功能完整，但逻辑与界面混杂，代码重复较多 |
| V2.0 分层重构版 | `login_v2.py` | 三层架构，封装数据库操作，业务与界面解耦 |

**推荐查看 V2.0 版本**，它最能体现本项目的核心价值：**用面向对象思想对混乱代码进行工程化重构的能力。**

## 功能列表
- 用户注册（自定义账号或系统随机生成）
- 用户登录校验
- 管理员密钥验证
- 管理员查看自身账号信息
- 管理员查看所有已注册用户
- 首次启动自动创建管理员账号

## 技术亮点
- **三层架构设计**：数据访问层 (DBHelper) → 业务逻辑层 (UserSystem) → 交互表现层 (main)，职责明确，互不越界
- **数据库封装**：统一数据库操作入口，消除代码冗余。未来切换数据库类型（如 MySQL）只需修改 DBHelper 一处
- **面向对象原则**：合理运用封装、分层思想，让代码具备良好的可维护性和扩展性
- **代码演进对比**：通过新旧版本的对比，直观展示了从“能跑”到“好改”的工程化过程
